# 文件
### **chattr** 文件隐藏属性
==chattr +ia attrtest==

> A：访问时不修改atime，防止过度访问磁盘 
> S: 从异步改为同步 
>a：只能增加数据，禁止修改删除 
>i：禁止删除，修改收
>s：如果被删除，彻底删除
>u：只删除note

### 搜索文件
==local [ir] keyword== 

>  -r   后面接正则表达式
>  -i  忽略大小写


----------


==whereis [-bmsu] keyword==

> -b 只找二进制文件
> -m 说明
> -s 找source源文件
> -u 不在上面三个选项中的
> 


----------
==find [PATH] [OPTION] [ACTION]==

> -mtime,-ctime,-atime
>**-mtime +-n**  n天之前的**”一天之内“**
> **-mtime +n**  n天之前
> **-mtime -n**   n天之内
> -newer file 比file这个文件更新的
> -name filename 查找文件名filename的文件
> -size [+-] 比size大或者小的文件，有c=byte k=bytes 例如-size +50k
> -perm mode 权限刚好等于mode的文件
> -perm -mode权限必须包含mode的文件
> -perm +mode 权限必须含有mode中的一项的文件
> -user name name为账号名称
> -nouser 没有所有权
